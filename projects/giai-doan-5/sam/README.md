# SAM

SAM là nền tảng hỗ trợ onboarding và mentor học tập, chia làm backend FastAPI và frontend React.

## Cấu trúc
- Backend: FastAPI + SQLAlchemy + Pydantic
- Frontend: React (sẽ triển khai ở bước sau)

## Bước tiếp theo
1. Cài đặt dependencies backend
2. Thiết lập cơ sở dữ liệu và models
3. Xây dựng API onboarding/mentor flow
